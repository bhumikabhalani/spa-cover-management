| `spa-cover-management-dashboard.png` | Dashboard |
| `spa-cover-management-customers.png` | Customers list |
| `spa-cover-management-orders.png` | Orders list |

| `spa-cover-management-cover-preview.png` | Cover preview |

## Location

```
storage/app/public/screenshots/
```

After running `php artisan storage:link`, files are also available at:

```
/storage/screenshots/{filename}
```

## Naming convention

```
spa-cover-management-{page-slug}.png
```

Use lowercase kebab-case slugs that match the Filament page or resource name.
